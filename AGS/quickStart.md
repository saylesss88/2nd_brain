# Quick Start

1. Install:

```bash
paru -S aylurs-gtk-shell-git
```

2. Initialize a project:

```bash
ags init
```

3. Run the project:

```bash
ags run
```

4. Learn [[TypeScript in Y minutes]].

5. Read the Astal Documentation to start developing.

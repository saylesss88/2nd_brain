# Set up a new machine with a single command

```bash
chezmoi init --apply https://github.com/TSawyer87/dotfiles.git
```

If you use GitHub and your dotfiles repo is called `dotfiles` this can be
shortened to:

```bash
chezmoi init --apply TSawyer87
```

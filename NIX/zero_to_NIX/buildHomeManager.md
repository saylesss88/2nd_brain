# Build Home Manager

```nix
nix build "https://flakehub.com/f/nix-community/home-manager/*"
```

Here, `https://flakehub.com/f/nix-community/home-manager/*` is a flake reference
to the `nix-community/home-manager` repo on FlakeHub.
